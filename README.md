# Employee Management Dashboard

A modern employee management application built with Vue 3 that enables users to manage employee records through a responsive and intuitive interface. The application supports viewing, creating, editing, and deleting employee information, along with filtering, sorting, and form validation features.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Vee Validate
- Zod

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
